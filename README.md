# 🌟 About The Project

**Ravixa AI Study Desk** is a premium AI-inspired productivity dashboard designed for students, developers, and professionals who want a distraction-free and visually engaging workspace.

The project combines modern UI/UX principles with interactive frontend features to create an elegant study environment using only **HTML, CSS, and Vanilla JavaScript**.

This project focuses on delivering a beautiful user experience through glassmorphism, animations, reusable components, and responsive layouts without using any frontend framework.
